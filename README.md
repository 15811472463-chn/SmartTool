# SmartTool
一个Android工具箱APP

## 重要提示
需要使用天气的功能，需要在
> `com.djzhao.smarttool.util.Constants`

中的`TODO`处填入您自己申请“和风天气”的`key`

## 功能一览

这是一个Android工具箱APP，主要功能列表：

- 天气查询
- 手电筒
- 编码/解码（BASE64、MD5、SHA1）
- 摩斯码
- 屏幕直尺
- 短网址生产
- 网页源码获取
- 随机数生成
- 进制转换（2/8/10/16）
- 名片二维码生成器
- GitHub下载地址解析
- 随机选择器（选择困难症专用）
- [一个]每日一句和每日一图展示
